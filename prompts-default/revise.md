# Revise

The context contains the WRITING CONSTITUTION, the SCOPE, the current DRAFT, and
an audit JSON listing findings. Apply every finding's `fix`, in place. Change
only what the findings call for plus any deterministic tell they imply (em dash,
contrastive negation, generic anchor text, stock phrase); do not rewrite clean
sections or restate unchanged content.

Resolve CRITICAL findings first. If a finding cannot be fixed without a fact you
do not have, insert a clearly marked placeholder rather than inventing one.

OUTPUT: the full revised article only (markdown). No preamble, no diff, no commentary.
